YouTrack Tools Documentation!
==========================================

YouTrack_ is an issue tracking system developed by JetBrains_.  It is highly
configurable and has a nice REST API.  `YouTrack Tools`_ is a Python_ package
for managing YouTrack_ via the REST API.
